Counts the number of each mapped value (equivalent to Map followed by CountValues).

```go
isEven := func(v int) bool {
    return v%2 == 0
}
lo.CountValuesBy([]int{1, 2, 2}, isEven)
// map[bool]int{false: 1, true: 2}
```
