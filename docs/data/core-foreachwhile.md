Iterates over elements of a collection and invokes the callback for each element until the callback returns false.

```go
numbers := []int64{1, 2, -9223372036854775808, 4}
lo.ForEachWhile(numbers, func(x int64, _ int) bool {
    if x < 0 {
        return false
    }
    fmt.Println(x)
    return true
})
// Output:
// 1
// 2
```
